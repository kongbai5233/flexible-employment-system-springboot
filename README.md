# 灵活就业服务平台（flexible-employment-system-springboot）

本项目为基于微信小程序与 Spring Boot 的前后端分离系统，
实现灵活就业场景下的订单发布、图片上传、订单管理与动态信息展示等功能。

本仓库用于项目代码存档与技术总结，不包含生产部署配置。

---

## 一、项目架构

前后端分离架构：

- 前端：微信小程序
- 后端：Spring Boot
- 数据库：MySQL
- 图片存储：服务器本地文件系统

系统结构：

MiniApp → REST API → Spring Boot → MySQL

---

## 二、核心功能

- 微信授权登录
- 订单发布与管理
- 图片上传与路径回传
- 订单状态更新
- 动态发布与互动
- 地址选择与区域联动

---

## 三、技术栈

### 前端
- 微信小程序原生框架
- WXML + WXSS
- 异步请求处理

### 后端
- Spring Boot
- RESTful API
- JDBC / JdbcTemplate
- Multipart 文件上传

### 数据库
- MySQL
- 订单表、用户表、雇佣关系表、地区表等

---

## 四、关键实现说明

### 1. 图片上传机制

- 使用 multipart/form-data 上传
- UUID 生成唯一文件名
- 返回图片访问路径供前端渲染
- 配置文件中调整上传大小限制

### 2. 订单提交逻辑

- 前端表单校验
- 后端参数接收与数据库写入
- 返回状态标识进行前端提示

### 3. 前后端分离设计

- 通过 REST API 实现数据交互
- 保持前端与业务逻辑解耦
- 提高系统可扩展性

---
